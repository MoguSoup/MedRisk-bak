# Brand assets

Runtime branding uses only the transparent PNG files in this directory.

- `hbut-logo.png` and `hbut-wordmark-cn.png` are processed from the official
  Hubei University of Technology identity resources:
  <https://www.hbut.edu.cn/xxgk/xxbs.htm>
- `zhuopu-logo-full.png` and `zhuopu-mark.png` are processed from the company
  logo images supplied in the local external-input directory.

The source backgrounds were removed, transparent margins were trimmed, and the
runtime images were resized for web delivery.

Local startup, CI, and portable conda `MedRisk` environment script changes do
not alter these image assets or their source attribution.
