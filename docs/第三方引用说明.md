当前重建版本以自研实现为主，没有整段复制未确认许可证项目代码。后续若直接移植外部源码，应将对应 LICENSE 文本和修改说明加入本目录。

本地启动、停止、CI、可移植 conda `MedRisk` 环境解析和 TypeScript 构建配置的脚本修复属于项目工程化维护，不新增第三方源码引用，也不改变上表许可证边界。

正式开源前仍需要在仓库根目录加入项目级 `LICENSE` 文件，并确认发布许可证与上表引用边界兼容。个人工作日志、实习报告、AI 工具记录和未确认授权的数据/图片/截图不属于第三方引用说明范围，也不得随源码发布。
